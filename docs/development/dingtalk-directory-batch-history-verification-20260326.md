# 钉钉目录批量处理历史验证

日期：2026-03-26

## 变更范围

- 批量结果从单条扩展为最近 `5` 条可回看
- 支持点击历史记录恢复查看旧结果
- 恢复后可继续复制失败清单和导出失败 CSV
- 支持一键清空最近批量处理记录
- 失败批次增加失败徽标与红色高亮
- 失败批次支持一键切换到失败成员视图
- 失败批次支持推荐下一步动作
- 失败批次支持失败原因分组摘要
- 失败原因摘要项可点击筛选对应失败成员
- 失败原因筛选后，推荐动作会自动收窄到当前失败原因子集
- 失败原因摘要项直接显示建议动作提示
- 激活失败原因后支持复制人工处理备注模板
- 激活失败原因后支持单独复制/导出该类失败清单
- 人工处理备注支持模板预设切换
- 人工处理备注支持上下文变量与快捷补充语
- 人工处理备注支持输出格式切换
- 失败原因备注上下文按原因隔离，并支持恢复推荐字段/清空补充说明
- 人工处理备注支持处理组预设
- 切换到新输出格式时会保留当前处理组草稿
- 人工处理备注支持团队标准模板
- 失败原因初始化会在个人偏好与团队标准之间按优先级收口
