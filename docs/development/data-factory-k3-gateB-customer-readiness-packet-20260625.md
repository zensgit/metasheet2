## 需要贵方提供（Gate B 输入清单）

### 1. K3 WISE WebAPI 访问
- WebAPI 服务地址（endpoint URL）。
- 服务账号凭据（账号/密码或 token）——**经安全渠道单独提供，勿邮件明文**。
- 鉴权/会话模式（authority code / token mode）及任何调用频率限制。

### 2. 网络连通
- 从部署主机到 K3 WebAPI 的网络放行（防火墙 / VPN / IP 白名单）。
- 若使用 SQL 通道：到 SQL Server 的连通同样放行。

### 3. SQL Server 访问（仅当使用 SQL 只读批量通道时）
- 只读账号 + 连接参数（host / port / database）。
- 执行器 / 代理在贵方部署侧就绪（本地部署侧职责；非我方交付）。

### 4. PLM 源系统（仅当 PLM 作为来源时）
- PLM 接入信息，并注册为外部系统（source 角色）。

### 5. 字段映射 + 字典规则
- 物料 / BOM 字段映射表（K3 字段 ↔ 台账字段）。
- 选项 / 字典规则（如计量单位、物料类别等枚举）。

### 6. 回滚 / 证据负责人
- 指定一名负责人，对任何写回签字确认，并负责回滚。
